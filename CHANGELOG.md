## [1.0.0] - 2026-07-14

### Added
- Initial public release of the full 12-stage prompt pipeline: system prompt, company context template, prospect discovery, lead research, qualification, outreach email, LinkedIn outreach, follow-ups, CRM notes, objection handling, meeting prep, proposal generation
- Full documentation set in `/docs` (getting started, installation, configuration, workflow, prompt guide, best practices, customization, examples, FAQ, troubleshooting)
- Seven fillable templates in `/templates` (ICP, company context, buyer persona, outreach framework, discovery questions, lead scoring, CRM)
- Six fully worked industry examples in `/examples` (SaaS, Agencies, AI Startups, Education, Healthcare, Manufacturing)
- Seven tool integration workflows in `/workflows` (Apollo, Clay, Instantly, Gmail, Notion, HubSpot, Salesforce)
- GitHub issue and pull request templates
- MIT License, Code of Conduct, Security Policy, Contributing Guide
